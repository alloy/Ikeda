    Fragaria 0.0.1 Reference

    What is it? Fragaria is a Cocoa syntax colouring NSTextView.

    Where did it come from? Fragaria is the vital pulp of Smultron (now Fraise).

    Fragria is implemented as a framework that supports both reference counting and GC.

    Thanks to Rui Pacheco for the RC changes.

    Other information regarding the original source may be availble: http://sourceforge.net/projects/smultron/files/smultron http://www.fraiseapp.com/
