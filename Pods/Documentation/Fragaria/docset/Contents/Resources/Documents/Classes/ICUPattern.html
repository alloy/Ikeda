Overview

@class       ICUPattern
@abstract    A compiled regular expression.
@discussion  <a href="http://icu.sourceforge.net/">ICU</a> provides a widely used Unicode

regular expression library. This class can be roughly mapped to the ICU C++ Pattern class. Notes about ICU regular expressions can be found at the ICU User Guide.
